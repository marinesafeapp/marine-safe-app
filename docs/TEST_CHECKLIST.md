## Test checklist (quick)

### Installation / startup

- [ ] APK installs successfully
- [ ] App launches without crashing
- [ ] Splash screen shows Marine Safe logo

### Permissions

- [ ] Notification permission prompt appears and can be granted
- [ ] Location permission prompt appears and can be granted

### Trips & overdue notifications

- [ ] Can start a trip and set an ETA
- [ ] Overdue notification fires when ETA is passed (while app is backgrounded)
- [ ] Overdue notification fires when app is swiped away / closed (Samsung reliability)
- [ ] Ending the trip stops further overdue alerts

### Settings reliability prompts (Android)

- [ ] “Open settings” buttons actually open system settings screens
